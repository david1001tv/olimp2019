## INSTALLATION:

**To run backend:**
- go to `/backend` directory and run `npm install`
- create your `config.js` and enter your configurations of DB
- run `npm run debug`

**To run frontend:**
- go to `/frontend` directory and run `npm install`
- run `npm start`
